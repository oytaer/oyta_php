# 调试模块 (Debug)

## 模块概述

调试模块提供 OYTAPHP 的调试和优化功能，包括 Trace 调试面板、SQL 日志、路由/配置缓存等。

## 模块结构

```
debug/
├── mod.rs          # 模块入口
└── trace.rs        # Trace 调试面板
```

## 功能特性

| 功能 | 说明 |
|------|------|
| Trace 面板 | 实时调试面板 |
| SQL 日志 | 数据库查询日志 |
| 路由缓存 | 路由缓存优化 |
| 配置缓存 | 配置缓存优化 |

## 使用示例

### 启用调试模式

```php
<?php
// .env
APP_DEBUG=true
```

### 访问调试面板

```
http://localhost:8000/_trace
```
