# 项目检测模块 (Project)

## 模块概述

项目检测模块提供项目检测和信息获取功能。

## 模块结构

```
project/
├── mod.rs          # 模块入口
├── detector.rs     # 项目检测器
└── info.rs         # 项目信息
```

## 功能特性

| 功能 | 说明 |
|------|------|
| 项目检测 | 检测项目类型 |
| 信息获取 | 获取项目信息 |
| 路径解析 | 解析项目路径 |

## 使用示例

### 检测项目

```bash
oyta init
```

## 技术实现

- 项目类型检测
- 项目信息获取
- 路径解析
