# 文件监视模块 (Watcher)

## 模块概述

文件监视模块使用 notify 库监听项目目录的文件变化，当 PHP 文件发生变化时，自动重新解析并更新符号表，实现热重载功能。

## 模块结构

```
watcher/
├── mod.rs          # 模块入口
├── watcher.rs      # 文件监视器
└── hot_reload.rs   # 热重载处理
```

## 功能特性

| 功能 | 说明 |
|------|------|
| 文件监视 | 监听文件变化 |
| 热重载 | 自动重新加载 |
| 符号表更新 | 更新解析结果 |

## 使用示例

### 启用热重载

```bash
# 启动服务器时自动启用热重载
oyta run --debug
```

### 监视目录

```
监视的目录：
- app/
- config/
- route/
- view/
```

## 技术实现

- 基于 notify 库
- 异步文件监视
- 增量更新
- 低性能开销
