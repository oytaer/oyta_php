# I18n 国际化模块

## 模块结构

```
i18n/
├── mod.rs    # 模块入口
└── i18n.rs   # 国际化实现
```

## 基本使用

```php
<?php
// 获取翻译
echo Lang::get('welcome');
echo Lang::get('user.not_found', ['name' => 'John']);

// 设置语言
Lang::setLocale('zh-cn');

// 获取当前语言
$locale = Lang::getLocale();

// 判断语言环境
if (Lang::is('zh-cn')) {
    // ...
}
```

## 语言文件

```php
<?php
// lang/zh-cn.php
return [
    'welcome' => '欢迎',
    'user.not_found' => '用户 :name 未找到',
    'validation.required' => ':attribute 不能为空',
];

// lang/en-us.php
return [
    'welcome' => 'Welcome',
    'user.not_found' => 'User :name not found',
];
```

## 配置

```php
<?php
// config/lang.php
return [
    'default' => 'zh-cn',
    'supported' => ['zh-cn', 'en-us'],
    'fallback' => 'zh-cn',
];
```
